Custom Modification Library:
    Input must be a tabulated file containing columns with these name:
  • 'Ytype' for Ytype modification,
  • 'type' for type of the modification,
  • 'mass' for mass of the modification,
  • 'regex' for regular expression of the modification,
  • 'label' for label of the modification,
  • 'm_label' for final label of the modification if it is a static modification on the generated sequence
Custom SWATH Windows Range:
Input must be a tabulated file where the first column is the starting mz value of the window and the second column is the ending mz value of the window
Each RT must be separated by newline.
